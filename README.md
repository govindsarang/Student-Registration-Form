# 📋 Student Registration Form

A simple, clean **Student Registration Form** built using pure HTML — no CSS, no JavaScript required! This project demonstrates the use of various HTML input types, semantic structure, and form accessibility best practices.

---

## 🌐 Live Preview

> Open `index.html` directly in any browser to see the form in action.

---

## 📁 Project Structure

```
Student-Registration-Form/
│
└── index.html        # Main registration form
```

---

## 📝 Form Sections

| Section | Input Type Used |
|---|---|
| Full Name | `<input type="text">` |
| Email Address | `<input type="email">` |
| Password | `<input type="password">` |
| Phone Number | `<input type="tel">` |
| Date of Birth | `<input type="date">` |
| Gender | `<input type="radio">` |
| Course Selection | `<select>` (dropdown) |
| Skills / Interests | `<input type="checkbox">` |
| Address | `<textarea>` |
| Student Photo Upload | `<input type="file">` |
| Terms & Conditions | `<input type="checkbox">` |
| Submit / Reset | `<button type="submit/reset">` |

---

## ✅ Features

- Uses semantic HTML with proper `<form>` structure
- Every input has a matching `<label>` for accessibility
- `name` attribute on all inputs (required for form data submission)
- `required` attribute for built-in HTML5 validation
- Supports different input types for different data
- Works without any external libraries or stylesheets
- Clean layout without unnecessary borders or icons

## 🚀 How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/govindsarang/Student-Registration-Form.git
   ```

2. **Navigate to the project folder:**
   ```bash

   ```

3. **Open in browser:**
